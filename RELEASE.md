# Device Database v4.17.0
For details, refer to [README.md](./README.md).

## Version info
* Add new PSoC 4100S Plus Automotive part numbers.
* Add new CCG7S part numbers.
* Add new TRAVEO™ T2G CYT4DN part numbers.
* Bug fixes.

## Recommended device support libraries
* mtb-pdl-cat1 >= 3.11.1
* mtb-pdl-cat2 >= 2.11.0 (new PSoC 4100S Plus and CCG7S part numbers are expected in upcoming 2.12.0)
* mtb-xmclib-cat3 >= 4.4.0
* mtb-pdl-cat5 1.0.0
