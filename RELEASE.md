# Device Database v4.19.0
For details, refer to [README.md](./README.md).

## Version info
* Add PSOC™ C3 device support.
* Update Bluetooth configuration data for ModusToolbox 3.3 BT Configurator.
* Update QSPI configuration data for PSOC™ 6 devices.
* Minor bug fixes.

## Recommended device support libraries
* mtb-pdl-cat1 >= 3.13.0
* mtb-pdl-cat2 >= 2.12.0
* mtb-xmclib-cat3 >= 4.4.0
* mtb-pdl-cat5 >= 1.3.0
